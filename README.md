# 📌 Project Overview

Python has become one of the most widely used programming languages in **Test Automation**, **API Testing**, **AI-assisted Testing**, and **DevOps**.

This repository contains a collection of practical Python programs and coding exercises that helped build my programming foundation as a QA Automation Engineer.

Although originally developed in **2023**, the concepts covered remain fundamental and highly relevant for automation engineers, SDETs, and software testers learning Python.

This project focuses on writing clean, readable, and beginner-friendly Python code while strengthening problem-solving and programming skills required for automation testing.
